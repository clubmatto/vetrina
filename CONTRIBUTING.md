# Contributing

We love every form of contribution. By participating to this project, you agree
to abide to the Club Matto [code of conduct](CODE_OF_CONDUCT.md).

## Getting Started

Each project in this monorepo has its own setup instructions. Please refer to
the individual project's README.md for specific guidance.

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

We welcome all contributions!
